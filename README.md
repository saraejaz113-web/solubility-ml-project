# solubility-ml-project
Machine learning project to predict solubility based on temperature.

## features 
- uses temperature in °C as the input data.
- gives output in the form of solubility (g/100mL).
- based on linear regression ML model.

## How it works:
This model learns from the data values of solubility and temperature. It predicts the solubility at new input temperature from the relationship learned previously.
